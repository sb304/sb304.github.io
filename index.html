Zombie Apocalypse Game

This is a simple top-down shooter game I built for my software development course. There are two levels: a drone level and a boss level. Gameplay mechanics include random spawning, heart points, and smooth movement.

Cell Society

In a group of 3, I built a cell automaton simulation program which supports five different automaton models. The Wa-Tor Predator-Prey model and Schelling's Segregation model are shown above.

SLogo

In a group of five, I built an integrated develpment environment capable of supporting a restricted subset of the Logo programming language. The program reads and interprets user commands and visually displays expressions using "turtle" graphics."

Google PageRank

For my math seminar on numerical methods, I wrote a simplified Matlab implementation of Google's PageRank algorithm and a research paper describing the statistical and numerical methods involved. Project was presented in front of class.

Vooga Salad

Project was done in a team of 12. For our final project in my software development course, we built a game authoring environment where designers could build a variety of 2D-style scrolling platformer games through visual elements. Program is also equipped with a game engine capable of loading and running these game files.

Free Energy Calculation Script

As a research assistant for a computational biology lab, I wrote shell scripts that carried out molecular simulations using the Amber Molecular Dynamics package. I also wrote a python script that used data from Amber output to calculate free energy differences by using the Multistate Bennet Acceptance Ratio method. Script is still being used by the lab today.

Haiku Audio and Video Chat

As a summer intern for Epic Systems, I designed and implemented audio and video chat functionality in Epic's Haiku app for the Android platform using Google WebRTC APIs. I also implemented a new authentication protocol for Haiku users to sign in and connect to the hospital server.
